# Tests

## Regular test

```shell
$ npm run test
```

---

## Coverage

This command runs all tests and outputs a report<sup>1</sup> about its coverage.

```shell
$ npm run test:coverage
```

<sup>1</sup> The report outputs at **coverage** folder.

---

## Watching files

```shell
$ npm run test:watch
```

---

## ESLint

```shell
$ npm run lint
```

---

# Build

```shell
$ npm run build
```

<sup>\*</sup> Builds at **dist** folder
