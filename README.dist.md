# Quickstart

## Install

```shell
$ npm install validocs --save
```

---

## Usage

_Typescript_ example:

Import and use a module:

```typescript
import CPF from 'validocs/cpf';

const cpf: CPF.Cpf = CPF.make();

console.log(`Generated CPF number: ${cpf}`);
```
